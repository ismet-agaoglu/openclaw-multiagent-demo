# openclaw-multiagent-demo

Multi-agent demo workspace for OpenClaw (ma-chef, ma-translator, ma-planner, ma-coder, ma-tester).

## Quick Start

```bash
# Install deps (Node.js)
npm ci

# Run demo
npm start

# Run tests
npm test
```

## Agent Protocol

See `agents/README.md` for the strict multi-agent protocol (no chatter, minimal diffs, API-like outputs).
