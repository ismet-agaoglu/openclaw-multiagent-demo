# Multi-agent Workflow (Demo)

## Flow
1) Chef → Translator (TR brief → EN prompt)
2) Chef → Planner (EN prompt → plan + tasks)
3) Chef → Coder (apply plan)
4) Chef → Tester (test + report)
5) Chef → Chef (synthesize)

## Example
- Task: “Add a CLI entry point to demo.js”
- Translator: EN prompt
- Planner: steps + tasks
- Coder: changes files
- Tester: runs tests
- Chef: summary + next steps
